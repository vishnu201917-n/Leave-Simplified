LeaveSimplified
Overview

LeaveSimplified is a full-stack web application designed to simplify employee leave management. The system allows users to securely apply for leave, track leave status, and manage leave-related information through an intuitive interface.

Features
User Authentication
Leave Request Submission
Leave Approval & Management
Leave History
Responsive User Interface
MongoDB Database Integration
